# LibWeb
LibWeb é um sistema web que simula venda de livros.

## Sobre
Desenvolvido em Java com o framework Spring Boot.

Segurança implementada utilizando Spring Security com implementação de Argon2 para salting de senhas.

Persistência implementada com Spring Data JPA e Hibernate.

Frontend com html/css/js com Thymeleaf.
